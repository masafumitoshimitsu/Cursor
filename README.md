# StarCafe サンプルサイト

スターバックス風のカフェ公式サイトをイメージした静的サイトのサンプルです。

- トップヒーロー、季節のおすすめ、メニュー、リワード、店舗検索、フッターの構成
- レスポンシブ対応（モバイル〜デスクトップ）

## 使い方（ローカルプレビュー）

ターミナルで以下を実行し、ブラウザで `http://localhost:5173` を開いてください。

```bash
cd /workspace
python3 -m http.server 5173
```

停止するには、`Ctrl + C` を押します。

## フォルダ構成

```
/workspace
├─ index.html
├─ styles/
│  └─ main.css
└─ scripts/
   └─ main.js
```
